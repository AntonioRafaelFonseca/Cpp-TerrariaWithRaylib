## Crafting System:
hold 's' and select item in bottom left corner with mouse with mouse or use left/right arrow keys\
Then hold 's' and click the crafting table to add the selected items\
After that hold the space key and click the crafting table and if the recipe matches one of the folowing you get the items
## Crafting recipes:

* 8x Wooden Planks = 4x Bark
* 8x Torches = 1x Wooden Plank + 1x Coal
